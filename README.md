# 浮笺 FloatNote

一张可以停在桌面上的中文浮动便签。

浮笺适合用来收拢每天散落在电脑前的任务、灵感和专注时间。它既可以像便签一样安静地停在屏幕一角，也可以打开完整工作台，查看今日、本周、灵感、专注和历史复盘。

## 你可以用它做什么

- 记录今天要做的事情
- 整理本周任务和组会事项
- 随手写下一点灵感、想法或随笔
- 开始一次专注计时，并留下专注记录
- 给任务设置不同颜色的优先级或自定义标签
- 在主窗口里查看今日速览、最近灵感和历史复盘
- 调整主题、字体和透明度，让便签更贴近自己的桌面

## 两种使用形态

### 桌面浮窗

浮窗适合轻量记录。它可以停在屏幕边缘或角落，像一张贴在桌面上的便签，用来快速添加任务、写灵感、看专注计时。

### 主窗口工作台

主窗口适合集中整理。你可以在这里查看今日任务、本周任务、最近灵感、专注统计和历史记录，也可以进入设置调整外观与偏好。

## 数据会保存在哪里

浮笺的数据默认保存在你自己的电脑本地。

别人下载并安装浮笺后，他们的任务、灵感、专注记录和历史复盘都会保存在他们自己的电脑里，不会读取你的数据，也不会上传到 GitHub。

## 当前状态

浮笺目前仍是早期版本，功能会继续迭代。后续计划包括更细致的视觉主题、数据导入导出、更完整的复盘视图，以及更顺手的桌面交互体验。

## 名字

“浮笺”取“浮在桌面上的纸笺”之意，希望它像一张轻巧的中文便签，安静、顺手，也有一点自己的气质。
