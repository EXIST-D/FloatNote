# 浮笺 FloatNote

浮笺是一款面向中文场景的桌面便签与轻量任务工作台。

它保留一个可停留在桌面的浮窗，用来快速记录今日任务、本周任务、灵感随笔和专注计时；同时提供主窗口，用来查看今日工作盘、任务列表、灵感记录、专注统计和历史复盘。

## 当前版本

当前公开版本：`v0.4.0`

Windows 用户可以在 GitHub Releases 中下载安装包：

```text
浮笺_0.4.0_x64-setup.exe
```

## 主要功能

- 桌面浮窗：便签质感，可用于快速记录和查看任务。
- 主窗口工作台：集中查看今日任务、本周任务、最近灵感、专注统计和历史记录。
- 今日任务与本周任务：支持新增、完成、编辑、删除、拖拽排序。
- 自定义任务等级：默认高、中、低三档，可自定义名称和颜色。
- 灵感记录：支持保存随笔，并可转为今日任务或本周任务。
- 专注计时：记录专注时长，结束后可写入历史。
- 历史复盘：支持今日/本周结束总结，并可删除误触记录。
- 外观设置：支持主题、字体、透明度、首页文案等配置。
- 系统托盘与快捷键：支持从托盘和快捷键快速打开。

## 数据存储说明

浮笺当前使用本地 SQLite 数据库。

每位用户安装并运行应用后，任务、灵感、专注记录和历史复盘都会保存在各自电脑的本地应用数据目录中。不同用户之间的数据互不相通，也不会上传到 GitHub。

除非后续主动加入云同步、账号系统、导入导出或共享数据库路径，否则用户数据默认只存在于用户自己的电脑上。
